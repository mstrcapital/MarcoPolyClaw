# 5分钟市场策略: 恐慌价差套利

> 来源: @k1rallik / @KyleDeWriter 推文
> 更新: 2026-02-17

## 案例: @MuseumOfBees

| 指标 | 数据 |
|------|------|
| 2天收益 | $52k (12x) |
| 起始 | $4.5k |
| 总收益 | +$75,220 |
| 胜率 | 73% |
| 交易数 | 1,073 |

## 策略核心

**"等群众恐慌买入，然后买对面"**

1. 当 UP 涨到 $0.90+，群众恐慌
2. DOWN 跌到 $0.10 以下
3. 买入便宜的 DOWN
4. 等待结算

## 数据支撑 (684个5分钟市场)

| 赔率 | 实际胜率 |
|------|----------|
| 5¢ (5%) | **8.8%** 获胜 |
| 10¢ (10%) | **15.6%** 获胜 |
| 价格不动 | 仅 0.6% |

## 执行策略

```
买 20 次 @ 5¢
├── 100% 至少1次获胜
├── 76% 概率第2次也赢
└── 输18次，但2次赢就覆盖
```

## 关键洞察

- **赔率 ≠ 概率**: 5%赔率实际8.8%胜率
- **高频**: 每天288个5分钟市场
- **低成本**: 每次只买几美分
- **高容错**: 输多也能赚

## 技术栈

- Rust (低延迟)
- London VPS (接近交易所)
- WebSocket 实时推送
